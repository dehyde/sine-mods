# Pinned Folder Colors

Pinned Folder Colors adds an eight-color palette to Zen Browser's native pinned folders. Each workspace palette is generated from that workspace's selected gradient colors.

Right-click a top-level pinned folder and open **Folder color** to choose from the 3×3 swatch matrix or restore Zen's default color. The saved choice is the slot, not a fixed color, so moving the folder to another workspace recomputes it from the destination workspace.

Nested folders inherit their family from the top-level folder. Siblings receive small hue shifts, while each deeper level receives a lightness and chroma shift.

## Settings

- **Color intensity** offers 0%, 20%, 40%, 60%, and 80% presets. The 40% preset is the default; 0% uses Zen's native folder strength.
- **Adjust generated colors for sidebar contrast** preserves workspace colors by default and adjusts only low-contrast colors when enabled.
- **Color Zen Folder Tree Connectors when that mod is enabled** tints the existing connector paths without creating new connectors.

Folders without a manual selection continue to use Zen's native folder colors.
