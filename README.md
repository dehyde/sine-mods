# Sine Mods

Canonical source repository for Tom Bar-Gal's Zen Browser Sine mods.

Each mod lives in `mods/<mod-id>/` with its own `theme.json`, scripts, styles, preferences, tests, and README. Submit a mod to the official Sine Store using the folder URL:

`https://github.com/dehyde/sine-mods/tree/main/mods/<mod-id>`
